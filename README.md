# Vue 3 Portfolio Website

**My first Vue.js project!** This portfolio website was built as a learning experience with Vue 3 and Vite. As a beginner to Vue.js, I created this project from scratch to understand the fundamentals of modern web development with Vue.

## About This Project

- **First time using Vue.js** - This is my introduction to the Vue ecosystem
- **AI for guidance only** - No AI was used to generate code. AI assistance was only used for asking questions and getting guidance when I was stuck
- **Learning focused** - Built to understand Vue 3 concepts, component architecture, and modern development workflows

## Technology Stack

- Vue 3 with `<script setup>` syntax
- Vite for development and building
- Vue Router for navigation

## Getting Started

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).
